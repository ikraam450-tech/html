1. accesskey – Defines a keyboard shortcut for an element.
  2. autocapitalize – Controls automatic capitalization on mobile devices.
  3. autofocus – Automatically focuses the element when the page loads.
  4. class – Specifies one or more class names for styling or JavaScript.
  5. contenteditable – Makes the element editable.
  6. data-* – Stores custom data for the element.
  7. dir – Sets text direction (ltr or rtl).
  8. draggable – Specifies whether the element can be dragged.
  9. enterkeyhint – Changes the Enter key label on virtual keyboards.
  10. exportparts – Exports shadow DOM parts for styling.
  11. hidden – Hides the element.
  12. id – Defines a unique identifier for the element.
  13. inert – Makes the element non-interactive.
  14. inputmode – Hints the type of keyboard to display.
  15. is – Specifies a customized built-in element.
  16. itemid – Provides a unique ID for microdata.
  17. itemprop – Defines a property for microdata.
  18. itemref – References related microdata elements.
  19. itemscope – Defines a new microdata item.
  20. itemtype – Specifies the type of microdata item.
  21. lang – Specifies the language of the element.
  22. nonce – Used for security with Content Security Policy.
  23. part – Allows styling of shadow DOM parts.
  24. popover – Creates a popover element.
  25. role – Defines accessibility role (ARIA).
  26. slot – Assigns content to a slot in web components.
  27. spellcheck – Enables or disables spell checking.
  28. style – Adds inline CSS styling.
  29. tabindex – Controls tab order for keyboard navigation.
  30. title – Provides extra information as a tooltip.
  31. translate – Specifies whether the content should be translated.
