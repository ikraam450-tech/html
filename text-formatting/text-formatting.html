1. Text Formatting

Bold text is used in advertisements.

Strong text shows important warnings.

Italic text is used for book titles.

Emphasized text adds stress in speech.

Highlighted text shows exam keywords.

H2O (Chemistry example).

102 = 100 (Math example).

$150 $100 (Online shop discount).

Explanation: Text formatting elements improve readability and meaning. Some tags like <strong> have semantic importance.

2. Character Entities

Less than: <

Greater than: >

Copyright: © 2026

Ampersand: AT&T

Explanation: Character entities allow us to display reserved symbols that HTML normally uses for coding.

3. Description List

HTML
Structures the content of a website.
CSS
Styles and designs the website layout.
JavaScript
Adds interactivity and dynamic behavior.
Explanation: <dl> creates the description list, <dt> defines the term, <dd> describes the term. Used in glossaries and FAQs.

4. Blockquote

"Education is the most powerful weapon which you can use to change the world."
Explanation: The <blockquote> element is used for long quotations from books, articles, or famous people.
